void LoadBalancer::performCycle ( )

Performs a single clock cycle for the load balancer.

This function iterates through all the servers and checks if any of them have completed their current task. If a server is available, it will start processing the next request in the queue. If no servers are available and the request queue is empty, it sets the serversDone flag to true.
